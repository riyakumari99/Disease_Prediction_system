## ✨ Features

* 🧠 Machine Learning-based disease prediction
* 📊 Probability-based results with graphs
* 🔐 User Login & Signup system
* 📜 History tracking of predictions
* 🤖 Health chatbot assistant
* 🏥 Nearby hospital locator (Google Maps)
* 👨‍⚕ Doctor suggestion system

---

## 🛠 Tech Stack

* **Frontend:** Streamlit
* **Machine Learning:** Scikit-learn (Naive Bayes)
* **Visualization:** Matplotlib

---

## ⚙️ How It Works

1. User selects symptoms
2. Model processes input
3. Predicts top 3 diseases with probability
4. Displays:

   * Description
   * Precautions
   * Doctor recommendation
